Here, features are columns that we are dependend upon to see the relation to the output which will be the target variable or columns. Here, we will choose Sepal and petal measurements as features to depend upon to see which target species it provides.

we need training data to train the model and testing data to test the result. so, train_test_split function is used to split the data here. We are using supervised machine learning here.

x = iris_dataframe[iris.feature_names] # features which is sepal and petal measurements
y = iris_dataframe["species"] # Target which is species, the relation of sepal and petal to know which species

# Splitting dataset to training and testing sets (80-train to 20-test)
x_train, x_test, y_train, y_test = train_test_split(x, y, test_size=0.2, random_state=42)

Training the model using Logistic Regression¶

In [12]:
# Train the model
# Using Logistic Regression (a simple supervised learning algorithm)
model = LogisticRegression(max_iter=200)
model.fit(x_train, y_train)

# Make predictions
y_pred = model.predict(x_test)

# Evaluate the model
# Accuracy Score
accuracy = accuracy_score(y_test, y_pred)
print(f"Model Accuracy: {accuracy:.2f}")

# Classification Report
print("Classification Report:")
print(classification_report(y_test, y_pred))

# Confusion Matrix
cm = confusion_matrix(y_test, y_pred)
sns.heatmap(cm, annot=True, cmap='Blues', fmt='d', xticklabels=iris.target_names, yticklabels=iris.target_names)
plt.title("Confusion Matrix")
plt.xlabel("Predicted")
plt.ylabel("Actual")
plt.show()

Explanation¶

We train the model using Logistic Regression which is simple yet effective supervised learning algorithm for classification based on species. The maximum iteration limit is initialized to ensure the model converges during training. We used .fit() method to train the model using training dataset we splitted earlier which learned patterns and relationship between input features (sepal and petal measurements) and the target labels which is flower species.

The trained model is used to make predictions using test data using the .predict() method. The predictions (y_pred) represented the species classification for the test samples based on the model understanding.

Finally, we evaluated the model;s performance using accuracy score which amounted to 1. This measures the proportion of correct predicions to the total number of test samples. The classification_report() provided the detailed report of model's precision, recall, fi-score and support for each species. This helps us understand how well the model performs for each category. Finally, the confusion matris is computed using confusion_matrix() and visualized it using the seaborn.
